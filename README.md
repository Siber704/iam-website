# Я Есть

Минималистичный медитативный сайт. Фраза медленно появляется, замирает и исчезает — приглашая к тишине.

## Что делает

Текст **«Я Есть»** проявляется по частям в медленном, дышащем ритме. Никаких кнопок, никаких отвлечений. Только присутствие.

Поддерживаемые языки: русский · английский · испанский

## Файлы

- `index.html` — структура страницы
- `style.css` — оформление (белый фон, текст по центру)
- `script.js` — логика анимации (появление / пауза / исчезновение)

## Как запустить

Открыть `index.html` в любом браузере. Сборка не нужна, зависимостей нет.

Инструкции по размещению на сервере — в `DEPLOYMENT.md`.
